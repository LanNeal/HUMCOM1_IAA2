How to Build a Computer

What you need to Build a Computer

  1. CPU
  2. Cooler
  3. Motherboard
  4. Memory/RAM
  5. GPU
  6. HDD/SSD
  7. Power Supply
  8. Computer Case
  9. Peripherals

    10. Instructions

    1. Gather what you need to build a Computer
    2. Open Case and prepare for installation
    3. Place the processor in the socket (Motherboard)
    4. Insert the Memory/Ram
    5. Install the Cooler
    6. Mount the motherboard into Case
    7. Connect the HDD/SDD
    8. Place Computer case fans
    9. Mount the PSU into case
    10. Plug the Power cables from the PSU into the motherboard, GPU, SSD, and CPU cooler
    11. Setup BIOS, Install OS, and update Drivers
    12. Ready to use
